# Ostiro Work v1 - Crousty Wok

Site vitrine consacré à **Crousty Wok** et à son univers de street food asiatique.

Ce projet est l'une de mes créations personnelles, imaginée et réalisée spécialement pour elle. Il présente simplement son identité, son histoire, sa carte et ses restaurants à travers une expérience visuelle moderne.

## Technologies

HTML, CSS et JavaScript.
